Community Contribution & Incentive Programs

How to Participate​

Based on your role, there are different ways to contribute to Aevir:

Developer​

  • Read the AI Tasks & Application Layer documentation
  • Integrate via APIs and build your own apps
  • Contribute to protocol development

Node Operator / Miner​

  • Learn how to become an execution node / training node / content node
  • Provide compute resources to earn Training Points
  • Provide storage to earn Library Points

Participant / Investor​

  • Understand $AEV / $veAEV design and long-term incentives
  • Participate in governance through veAEV voting
  • Lock AEV for governance rights and AUV minting

Incentive Mechanisms​

  • IntelliPoints: Quantified contribution scores (Training Points + Library Points)
  • Block Rewards: Distributed based on contribution and role
  • veAEV Benefits: Governance power and AUV minting rights
  • Reputation Boost: High contributors receive voting coefficient bonuses
