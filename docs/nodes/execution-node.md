# Execution Node

Execution Nodes form the high-performance engine of Aevir, responsible for processing all transactions and smart contracts.

## Responsibilities

- Participate in Execution Layer BFT consensus
- Process transactions and smart contracts
- Execute AI task scheduling and contribution accounting
- Maintain high performance for the network

## Requirements

- **Stake**: Sufficient $AEV to rank within the top 21 by stake amount
- **Hardware**: High-performance servers with strong compute capability
- **Network**: High bandwidth and low latency connectivity
- **Uptime**: Near 100% availability expected

## Rewards

- Share of Execution Layer block rewards
- Portion of transaction fees (as defined by tokenomics)

## Risks

- **Slashing**: Double-signing or prolonged downtime can result in stake loss
- **Competition**: Must maintain top 21 stake ranking to remain in committee

## Getting Started

1. Ensure you meet hardware and stake requirements
2. Set up node software following official documentation
3. Register your node on-chain
4. Begin participating in consensus
