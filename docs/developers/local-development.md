# Node Deployment & Local Development

## Local Development Environment

### Prerequisites

- Node.js 18+ or compatible runtime
- Docker (optional, for containerized setup)
- Git for version control

### Setting Up

1. Clone the Aevir development repository
2. Install dependencies
3. Configure local network settings
4. Start local development node

### Development Tools

- **Aevir CLI**: Command-line interface for node operations
- **Local Testnet**: Run a local Aevir network for testing
- **Block Explorer**: Local explorer for debugging

## Node Deployment

### Testnet Deployment

1. Obtain testnet tokens from faucet
2. Configure node for testnet
3. Register node on-chain
4. Begin participating in network

### Mainnet Deployment

1. Ensure all requirements are met
2. Stake required $AEV
3. Complete hardware attestation
4. Register and activate node

## Debugging & Monitoring

- Use logging and metrics for troubleshooting
- Monitor node health and performance
- Set up alerts for critical issues
